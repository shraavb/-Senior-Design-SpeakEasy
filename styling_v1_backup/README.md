# SpeakEasy v1 Color Palette Backup

This folder contains the original v1 styling/color palette for reference and potential reversion.

## How to Revert

To revert to the v1 color palette, copy the `index.css` file from this folder to `/src/index.css`:

```bash
cp styling_v1_backup/index.css src/index.css
```

## Design Philosophy (V1)

The v1 palette featured:
- **Cool blue tones** for backgrounds and UI elements
- **Purple accents** for the Social module
- **Cyan/teal** for the Tourism module
- **Orange** for the Professional module
- **Clean, modern feel** with blue-based neutrals
