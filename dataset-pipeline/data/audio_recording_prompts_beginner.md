# Audio Recording Prompts - BEGINNER Level (A1-A2)

**Project:** SpeakEasy - Catalan Spanish Language Learning App
**Purpose:** Collect learner audio samples for evaluation
**Target Speakers:** All newbie speakers (4 people)

---

## Recording Instructions

1. **Record both speakers** - You can do both parts yourself or record with a partner
2. **Speak at your current level** - mistakes are valuable data!
3. **Read the situation first**, then have the conversation naturally
4. **Pause briefly between speaker turns** (1-2 seconds)
5. **Record in a quiet environment** if possible

---

## How to Save Your Recordings

### File Naming Format
```
{your_name}_{category}_{dialogue_number}.wav
```

### Examples
```
john_social_01.wav
john_travel_02.wav
maria_business_03.wav
```

### Folder Structure
```
recordings/
├── john/
│   ├── john_social_01.wav
│   ├── john_social_02.wav
│   └── ...
```

### Recording Settings
- **Format:** WAV or MP3
- **Sample Rate:** 44.1kHz (default is fine)
- **Duration:** Each dialogue should be 30-60 seconds

## SOCIAL Dialogues (6 dialogues)

### Social 01 - Meeting a Friend at a Cafe
**Situation:** You run into a friend at a cafe and chat briefly.

| Speaker | Line |
|---------|------|
| A | "¡Hola! ¿Cómo estás?" |
| B | "¡Hola! Estoy bien, gracias. ¿Y tú?" |
| A | "Muy bien. ¿Qué haces aquí?" |
| B | "Estoy tomando un café. ¿Quieres sentarte?" |
| A | "Sí, gracias. ¿Cómo está tu familia?" |
| B | "Están bien, gracias. Mi madre está muy contenta." |
| A | "¡Qué bien! Me alegro." |

**Target words:** hola, cómo estás, bien, gracias, qué haces, familia

---
