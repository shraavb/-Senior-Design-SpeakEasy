# Audio Recording Prompts for Ground Truth - Set 2

**Project:** SpeakEasy - Language Learning App
**Purpose:** Native speaker audio samples for ground truth evaluation
**Source:** Dialogues extracted from our film dataset

---

## Recording Instructions

1. **Speak naturally** - as you would in everyday conversation
2. **Use your regional accent** - Catalan-influenced Spanish is perfect
3. **Record in a quiet environment** if possible
4. **Record the full dialogue exchange** - each line separately or as a conversation

## SOCIAL Dialogues (Personal & Social Module)

### 1. Reunion - Running Into an Old Friend
**Film:** A Gun in Each Hand (2012)
**Timestamp:** 00:01:10 - 00:02:12
**Scenario:** Two old friends unexpectedly run into each other after many years.
